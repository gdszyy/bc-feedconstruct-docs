Message Syntax

All messages are in JSON format. For all the messages that are sent or received, first 4 byte (binary integer) is the length of the message and the rest is the message in JSON format.

The format of the message is

{
    "Command": "Command Name",
    "Params": [
        {
            "param name": "value"
        },
        {
            "param name": "value"
        },
        {
            "param name": "value"
        },
        {
            "param name": "value"
        }
    ]
}

Where "Command Name" is one of our supported commands (See the list below) and "value" is a parameter value for current command.
