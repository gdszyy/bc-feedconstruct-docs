SubscribeMatches

Subscribing for all live Matches.

Note:
This method should be called once when client starts.
{
    "Command": "SubscribeMatches",
    "Params": [
        {
            "IncludeMatches": false,
            "IncludeMarkets": false,
            "GetChangesFrom": "2016-12-23T05:00:15.4303643Z"
        }
    ]
}



IncludeMatches -- return all Live matches IncludeMarkets -- include all Markets objects with Match GetChangesFrom -- return all changes starting from this date All fields are optional. If 'GetChangesFrom' is provided partner will get data not as 'Object' in Response, but as a separately update.
{
    "Command": "SubscribeMatches",
    "ResultCode": 0,
    "Objects": []
}
Possible Result Codes: 0, 11, 13, 18, 108


Object -- Array of subscribed live matches
