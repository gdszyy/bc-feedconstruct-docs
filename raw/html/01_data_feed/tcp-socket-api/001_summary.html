Summary

This document is intended for FeedConstruct Clients having own trading platform and willing to offer the end users (players) a comprehensive data on over 120 sports types.

The in-house produced Data feed is a product of easy integration through JSON. This document describes how the client should implement the integration and the functionality that we provide via API.

The API is updated on a regular basis, as we constantly add new features to it.
