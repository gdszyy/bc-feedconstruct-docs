Web Methods

Token
Access methodAddressResultParams
POST

*Token should not be requested more than once per 24 hours.
/api/DataService/Token
{
    "Params": [
        {
            "UserName": "username",
            "Password": "password"
        }
    ]
}
{
    "Token": "token",
    "ResultCode": 0
}

OR

CommandResponse
{
    "Error": {
        "Key": "InvalidUsernamePassword",
        "Message": "Invalid Username and/or password"
    },
    "Objects": [],
    "ResultCode": 15
}





Possible Result Codes: 0, 11, 15		Mandatory:
• UserName • Password

Book
Access methodAddressResultParams
POST

*Not more than 100 objects can be booked with one command.
/api/DataService/Book?token=[token]
{
    Params: [
        {
            "ObjectId": ObjectId,
            "ObjectTypeId": ObjectTypeId,
            "IsSubscribed": true or false,
            "IsLive": true or false,
            "SportId": Sport Id,
        },
    ],
}
CommandResponse


Where ‘Objects’ field contains list of ‘Book’ models.

Possible Result Codes: 0, 11, 12, 14, 17, 18, 106, 109*		Mandatory:
• token • ObjectId • ObjectTypeId • IsSubscribed • IsLive • SportId (only for region)
