GetClientDetails

The GetClientDetails method is called by FeedConstruct when the Partner calls either CreateBet or GetMaxBetAmount.

It uses the AuthToken generated by the Partner and returns the player`s data if the token is valid and the player is logged in to the Partner’s website.

If the player is not logged in, the method must return ErrorCode 1008. For other issues—such as an expired or invalid token—relevant error codes should be returned, as specified in the Error Codes section.

Important:The client`s currency must match one of the supported values listed in the Currency Codes section.

Important:The client`s currency should remain unchanged throughout the client’s lifecycle.

URL: /GetClientDetails

Example: http://partnerhostname:port/api/GetClientDetails

Where http://partnerhostname:port/api is the Partner’s Base URL

Method: POST
