## 文档正文
All games must start on the scheduled date for bets to stand/have action. A game is considered completed for settlement/calculation purposes if 50 minutes have been played.

All match markets will be settled on the score at the end of regulation time and will exclude overtime if played, unless stated otherwise.

Match Totals Odd or Even

If there is no score all bets will be settled as Even.

Period Betting

The relevant period must be completed for bets to stand/have action, unless the specific market outcome is already determined. Scores/action within that period only count for betting purposes.

Match Winner

This 2 way market will include overtime where played and any other way used to determine a winner.

Overtime markets

Only scores/action within overtime count for betting purposes.

If a match venue is changed then bets already placed will stand providing the home team is still designated as such. If the home and away team for a listed match are reversed, then bets placed based on the original listing will be void.
