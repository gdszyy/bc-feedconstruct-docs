## TRANSLATIONS API

### Version 0.10

Summary

Translations API is designed for providing content translations to Odds Feed partners. Only already integrated Clients can use this Web API.

Odds Feed service is supporting only english language and this service will give an opportunity to Odds Feed partners to build their own multilingual portals and apps in over 20 languages.
