# P02 — 体育 / 联赛页

## 目的

按运动、地区、联赛三层筛选展示赛事列表，支持赛前/今日/未来/滚球切换。

## 数据来源

| 区域 | 来源 |
|---|---|
| 左侧分类树 | M03 |
| 中部赛事列表 | M04 + M05（默认市场） |
| 顶部 tab 切换（赛前/滚球/收藏） | M11 + 本地状态 |

## 渲染策略

- RSC：分类树 + 列表骨架（按 sportId 缓存）
- Client：默认市场赔率订阅

## 关键组件

- `<CategoryTree/>`（M03）
- `<MatchList/>`（M04，虚拟化）
- `<MatchRow market="default"/>`（M05）

## 验收要点

- 切换运动不重拉描述（M12 复用）
- 滚球 tab 显示 producer 健康（M15）
- 列表中订阅范围随可视区动态调整
