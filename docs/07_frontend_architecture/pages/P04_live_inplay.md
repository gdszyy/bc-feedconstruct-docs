# P04 — 滚球大厅

## 目的

集中展示正在进行的赛事，强调实时性、密度与稳定性，是恢复机制（M10）与降级（M15）最容易暴露问题的页面。

## 数据来源

| 区域 | 来源 |
|---|---|
| 运动 / 联赛快捷过滤 | M03 |
| 滚球列表（比分、阶段、主市场赔率） | M04（filter live）、M05 |
| Producer 健康 | M15 |

## 渲染策略

- 客户端为主，进入即建立 WS 范围订阅（按可见 match）
- 列表虚拟化，订阅随滚动动态调整

## 关键组件

- `<LiveFilters/>`
- `<LiveMatchList/>`（M04）
- `<LiveMatchRow/>`（M04 + M05，仅主市场）
- `<HealthChip/>`（M15）

## 验收要点

- 大列表下 60fps 平滑
- producer down 时整页降级提示，下注禁用
- 订阅范围动态收敛，离开列表自动 unsubscribe
