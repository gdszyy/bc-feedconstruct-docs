Multiple Bets

- Max Odd for Multiple Bet

Max odd for multiple bet – the maximal possible odd for multiple bet. Generally, it is set 1000, and no matter the event count and odds of the selections in the bet, the total odd cannot be higher than the value set for “Max odd for multiple bet” section.

- Selection Count in Multiple Bet

Selection Count in Multiple Bet – the maximal possible event combination for multiple bets. If this section is set 50, the client will not be able to combine 51 or more events in one bet.
