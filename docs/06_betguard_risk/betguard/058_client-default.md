Client Default

- Live Selection Limit (%)

Live Selection Limit – is the default client live limit, which is calculated from Selection Liability. If there are not any values set, the default value is 100%.

- PreMatch Selection Limit (%)

PreMatch Selection Limit – is the default client pre-match limit, which is calculated from Selection Liability. If there are not any values set, the default value is 100%.

There is an option, which allows us to set individual Selection limits depending on the client`s sportsbook profile. For example, VIP clients can get default 150% Selection Limits, while Arbitrage Betting Clients can get 5% default Selection Limits.
